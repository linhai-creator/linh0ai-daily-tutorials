# 1. COMPLEMENTARY 🔴🟢 — 2 màu đối lập trên color wheel
#    Vibe: dynamic, contrast, energetic
#    Vd: đỏ + xanh lá, cam + xanh dương, vàng + tím
#    Keyword: "complementary color palette", "red and green contrast"

# 2. ANALOGOUS 🟡🟠🔴 — 3 màu cạnh nhau trên color wheel
#    Vibe: harmony, calm, natural
#    Vd: vàng + cam + đỏ, xanh + xanh ngọc + xanh dương
#    Keyword: "analogous color palette", "warm yellow-orange-red harmony"

# 3. MONOCHROMATIC 🔵🔷💙 — 1 hue với nhiều tone/sắc độ
#    Vibe: minimal, sophisticated, mood
#    Vd: các sắc xanh, các sắc đỏ, các sắc xám
#    Keyword: "monochromatic blue palette", "single hue variations"

# 4. TRIADIC 🔴🟡🔵 — 3 màu cách đều trên color wheel (tam giác)
#    Vibe: vibrant, balanced, festival
#    Vd: đỏ + vàng + xanh, cam + tím + xanh lá
#    Keyword: "triadic color palette", "red yellow blue triangle"
