# ═══════════════════════════════════════════════════════
# 💡 TIPS NÂNG CAO
# ═══════════════════════════════════════════════════════

# 1. Negative prompt > Quality tags về độ hiệu quả
#    → Thêm 5 negative phổ biến = ảnh đẹp hơn 30%
#    → Thêm 10 quality tags = ảnh đẹp hơn 10%
# 
# 2. Quality tags có "thứ tự ưu tiên"
#    → Đặt tag quan trọng TRƯỚC
#    → "masterpiece, 4K" > "4K, masterpiece"
# 
# 3. Đừng quá tải negative prompt
#    → 10-25 từ là sweet spot
#    → 50+ từ → AI confused, có thể tệ hơn
# 
# 4. Save snippet trên máy
#    → Dùng AutoHotkey, Espanso, hoặc Notes
#    → 1 click paste = tiết kiệm 5 phút mỗi ngày
