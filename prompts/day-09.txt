# 🎯 Day 9 — Prompt Engineering Nâng Cao
# Demo qua chủ đề: Áo dài Việt Nam (Truyền thống vs Cách tân)
# Test 12 ảnh: 3 cấp độ × 2 variation × 2 model

# ═══════════════════════════════════════════════════════════
# 📚 3 KỸ THUẬT NÂNG CAO SẼ HỌC HÔM NAY
# ═══════════════════════════════════════════════════════════

# 1. WEIGHTED SYNTAX — đặt trọng số cho keyword
#    (keyword:1.4)   → tăng trọng số (model nhấn mạnh hơn)
#    (keyword:0.7)   → giảm trọng số (giảm ảnh hưởng)
#    Range 0.5-1.5 là sweet spot, vượt quá dễ "burn"

# 2. PROMPT CHAIN — chuỗi prompt giữ character consistency
#    "Same person as previous, now wearing... in [setting]"
#    Bắt buộc dùng kèm reference image (Seedream 4.5 hỗ trợ 14 ref)

# 3. NEGATIVE WEIGHTED + LAYERED
#    Tách negative thành 2 lớp:
#    - Layer 1 (kỹ thuật): plastic skin, low quality, blurry
#    - Layer 2 (concept): (kimono:1.5), (hanbok:1.5), (qipao:1.5)
#    → Quan trọng cho ảnh áo dài vì model dễ nhầm sang trang phục Á khác

# ═══════════════════════════════════════════════════════════
# 💰 GIÁ + SETTINGS CHUẨN (cho cả 12 ảnh)
# ═══════════════════════════════════════════════════════════
# - Aspect ratio: 3:4 (chân dung dọc)
# - Resolution: 2K
# - Seedream 4.5: 350 credit/ảnh × 6 ảnh = 2,100 credit
# - GPT Image 2: 900 credit/ảnh × 6 ảnh = 5,400 credit
# Tổng: 7,500 credit (~7.5k VND, ~0.7% gói Ultra Member)
