## 📦 Format mỗi bài

Mỗi `day-XX.md` đều theo template:

- 🎯 **Mục tiêu** — Hôm nay bạn học được gì
- 🟢🔵🟣 **Level** — Bài này phù hợp với ai
- 📖 **Nội dung chính** — Lý thuyết + ví dụ
- 🎬 **Video demo** — Clip ngắn 30-60s minh họa thao tác
- 💡 **Prompt mẫu** — File `.txt` hoặc `.json` trong `/prompts/`
- ⚡ **Thử thách hôm nay** — Bài tập nhỏ để bạn tự làm
- ❌ **Lỗi thường gặp** — Cách tránh & fix
- ➡️ **Bài tiếp theo**
